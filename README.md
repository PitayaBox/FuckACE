# FuckACE
## FuckACE是一个用来优化所有使用ACE的游戏的免安装软件，可以一键设置ACE为效率模式，降低ACE的CPU亲和性，以降低ACE对电脑的性能影响。

## 核心机制
FuckACE通过监控和限制下列进程的CPU亲和性、进程优先级、效率模式、I/O优先级和内存优先级等参数，来实现对ACE进程的限制。
### SGuard64.exe
### SGuardSvc64.exe
### Weixin.exe
